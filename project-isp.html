RL ISP Auto‑Tuning

Agents (PPO/SAC) learn to tune ISP parameters toward studio references with metric‑driven rewards.

PyTorch Gymnasium OpenVINO Optuna

Overview

This project builds a reproducible RL pipeline where an agent selects ISP parameter updates and receives a reward based on image similarity to studio references (SSIM/PSNR + edge/color terms). The environment samples image patches for faster training and more robust learning.

Key Results

  • +0.05–0.10 SSIM over baseline on validation.
  • Patch‑based training → faster iterations, stable rewards.
  • OpenVINO accelerates evaluation throughput.

Tech

  • PyTorch, Gymnasium, Stable‑Baselines3 (PPO/SAC)
  • OpenVINO GenAI for fast inference
  • Optuna for hyperparameter sweeps
