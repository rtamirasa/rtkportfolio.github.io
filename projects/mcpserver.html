MCP Server — AI-Powered Diagnostic Platform

Modular diagnostic platform integrating local LLMs and reinforcement learning pipelines to automate system analysis, improve reliability, and enable secure offline operation.

Highlights

  • Architected a modular MCP platform to orchestrate diagnostic tools and automate workflows.
  • Integrated local LLM inference (Qwen + OpenVINO) with LangChain tool routing.
  • Enabled offline AI-powered diagnostics with verification and transport switching.
  • Built reproducible benchmarking pipelines for PPO, SAC, and CNN models.
