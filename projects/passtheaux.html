PassTheAux · Collaborative Music Queue

PassTheAux is a group music experience web app that allows users to host or join sessions, recommend songs, and vote on the queue. With integrations for Spotify and Apple Music, groups can enjoy a customized and democratic listening experience.

Highlights

  • Host or join music sessions with a clean React frontend.
  • Recommend songs and vote on the live queue (upvote/downvote).
  • Algorithms suggest tracks based on recently played songs.
  • Customizable group modes: central DJ, collaborative, or casual listening.
  • Spotify API & Apple Music API integration for direct playback.
  • Firebase for session data and authentication (with 2FA for hosts).

Stack

React TypeScript Firebase Spotify API Apple Music API Java Backend
