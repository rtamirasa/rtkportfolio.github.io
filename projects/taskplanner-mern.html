Task Planner (MERN) · CS 348 Project

Task Planner is a responsive full-stack web application developed as part of CS 348. It allows users to manage tasks with categories, due dates, and statuses while providing visual reports (bar & pie charts) for better productivity insights.

Highlights

  • Create, update, and delete tasks with fields like title, due date, category, and status.
  • Status tracking: mark tasks as Completed, In Progress, or Pending.
  • Dynamic bar and pie chart visualizations with time range filters (all-time, past week).
  • Responsive Material-UI design for seamless experience across devices.
  • Deployed on GCP (backend) and Vercel (frontend).

Stack

MongoDB Express React Node.js Material-UI Charts GCP Vercel
