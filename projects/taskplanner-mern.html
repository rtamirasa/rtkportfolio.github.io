Task Planner (MERN)

Full-stack task tracking app with CRUD, status tracking and visual reports (bar/pie charts) for categories and time ranges.

Highlights

  • Tasks: create/update/delete with title, due date, category, status.
  • Status: Completed / In-Progress / Pending with checkboxes.
  • Visual reports: dynamic bar & pie charts with filters.
  • Responsive UI with Material-UI.
Source: repo README.

Stack

MongoDB Express React Node MUI
